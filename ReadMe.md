# 📊 Telecom Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview

This Power BI dashboard analyzes customer churn in a telecom company using the IBM Telco Customer Churn dataset. The dashboard helps identify factors influencing customer churn and provides business insights for customer retention.

---

## 📈 KPIs

- Total Customers
- Churn Customers
- Churn Rate
- Average Monthly Charges
- Average Tenure
- Total Revenue

---

## 📊 Dashboard Features

- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Churn by Gender
- Churn by Senior Citizen
- Churn by Tenure Group
- Churn by Monthly Charges

---

## 🎯 Key Insights

- Month-to-month contract customers have the highest churn.
- Fiber optic customers show higher churn.
- Electronic check users churn more frequently.
- New customers are more likely to churn during their first year.
- Higher monthly charges are associated with increased churn.
